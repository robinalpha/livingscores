# Scores Vivants — v2.4.0

Six decks de pratique incarnée :
- **Avec le vivant** — rencontres sensorielles avec le non-humain (25 cartes)
- **Terrains** — jeu de paysage, mouvement et écriture (31 cartes)
- **Traversées** — scores d'improvisation dans le vivant (32 cartes)
- **Choreospores** — scores somatiques de Maija Hirvanen (31 cartes)
- **Milieux** — deck éco-somatique (38 cartes)
- **Territoires croisés** — acteurs de territoire, trois phases (23 cartes)

**Total : 182 cartes**

## Déploiement

```bash
npm install
vercel
```

Ou GitHub + Vercel pour déploiement automatique.

## PWA

Une fois déployé, installable comme application mobile :
- iOS : Safari → Partager → Sur l'écran d'accueil
- Android : Chrome → Menu → Installer l'application
